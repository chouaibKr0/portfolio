## 🌐 Deployment to GitHub Pages

### Option 1: Quick Deploy (Recommended)

1. Create a new repository named `chouaibKr0.github.io` (your username)
2. Push all files to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio deployment"
   git branch -M main
   git remote add origin https://github.com/chouaibKr0/chouaibKr0.github.io.git
   git push -u origin main
   ```
3. Your site will be live at `https://chouaibKr0.github.io`

### Option 2: Project Page

1. Push to any repository
2. Go to **Settings** → **Pages**
3. Select `main` branch as source
4. Site will be at `https://chouaibKr0.github.io/repository-name`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #06b6d4;
    --accent: #f59e0b;
    /* ... more colors */
}
```

### Content
Update personal information in `index.html`:
- Hero section text
- About section
- Projects and skills
- Contact information

### Profile Image
Replace the placeholder icon with your photo:
1. Add your image to the folder
2. Update the `.profile-placeholder` div in `index.html`
